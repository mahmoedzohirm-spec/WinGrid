# إضافات متقدمة: تقارير وإعدادات

لقد أضفت نقاط دخول (APIs) وتمثيلات واجهات لإدارة التقارير والإعدادات:

## ما تم إضافته
- API: `/api/admin/reports` — تقرير مبيعات أساسي (إجمالي الطلبات، الإيرادات عبر وظيفة RPC، مشتريون الأكثر)
- API: `/api/admin/settings` — قراءة/تحديث إعدادات المسابقة (سعر البطاقة، إجمالي البطاقات)
- مكون: `components/Admin/SettingsForm.tsx` — واجهة لحفظ الإعدادات من لوحة الإدارة

## اقتراحات للتطوير الإضافي
1. إضافة مصادقة Admin قوية (JWT + Roles)
2. إضافة تقارير زمنية (مبيعات يومية/أسبوعية/شهرية)
3. إضافة تصدير CSV / PDF للتقارير
4. إضافة إشعارات عند قبول الطلبات (SMS/Email)
5. إضافة لوحة تحليلات Grafana أو Metabase متصلة بقاعدة البيانات

إذا أحببت أبدأ بتطبيق أحد الاقتراحات أعلاه الآن — أي واحد تفضل؟
